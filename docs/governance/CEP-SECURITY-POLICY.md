## 1. Vulnerability Disclosure & Reporting

Security researchers and engineers discovering potential vulnerabilities in CEP are requested to submit reports privately to `security@cep-platform.org`.

---

## 2. Response SLAs & Security Patch Targets

- **Critical Severity**: Triage within **24 hours**; hotfix release within **72 hours**.
- **High Severity**: Triage within **48 hours**; patch release within **7 days**.
- **Medium/Low Severity**: Addressed in the next scheduled PATCH release.
