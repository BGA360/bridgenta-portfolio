## 1. Support Phases

1. **Active Support Phase**: Full feature development, MINOR version releases, security patches, and active engineering support (Minimum 18 months).
2. **Maintenance Support Phase**: Critical bug fixes, security patches, and compliance updates only (12 months following Active Support).
3. **Security Support Phase**: Critical security patches only (6 months following Maintenance Support).
4. **End-of-Life (EOL)**: Product version is no longer supported or patched.
