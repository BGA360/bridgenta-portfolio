## 2. Severity Classifications

* **Critical:** Page offline, security credential leak, or severe script failure. Blocks deployment.
* **High:** Missing core section, incorrect quantitative claims, major visual breakage. Blocks deployment.
* **Medium:** Minor layout wrap issue, metadata problem, cache delay. Must be resolved pre-launch.
* **Low:** Spacing inconsistencies, minor grammatical typos with no impact on meaning.
